# skin-cancer-classification-efficientnetb0

Project ini mengimplementasikan model deep learning untuk klasifikasi kanker kulit menggunakan dataset HAM10000. Model ini menggunakan EfficientNetB0 dengan teknik transfer learning dan optimasi hiperparameter menggunakan Bayesian Optimization

Dataset: 
Dataset yang digunakan dalam penelitian ini adalah HAM10000 (Human Against Machine with 10,000 training images). 
Dataset ini diperoleh dari dua sumber utama untuk memastikan integritas data: Harvard Dataverse (sebagai sumber orisinal dengan DOI: 10.7910/DVN/DBW86T) dan Kaggle Skin Cancer MNIST. Dataset ini terdiri dari gambar lesi kulit yang digunakan untuk melatih model klasifikasi berbasis arsitektur EfficientNetB0
